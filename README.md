# Projet Site Web – L1 Informatique

Projet réalisé en L1 Informatique dans le cadre de mon apprentissage du développement web.

## Objectif
Réaliser mon tout premier site web afin de comprendre le fonctionnement et le rendu d’un site côté navigateur.

Ce projet avait pour but de découvrir les bases du développement web et de visualiser concrètement l’impact du code sur l’interface utilisateur.

## Technologies utilisées
- HTML
- CSS
- JavaScript (bases)
- XML

## Description
Le site a été développé uniquement avec des technologies front-end, sans framework ni backend.
L’accent a été mis sur :
- la structure des pages HTML
- la mise en forme avec CSS
- quelques interactions simples en JavaScript

## Ce que j’ai appris
- Structurer une page web en HTML
- Styliser une interface avec CSS
- Comprendre le rendu visuel d’un site web
- Ajouter des interactions simples avec JavaScript
- Organiser un premier projet web

## Statut
Projet universitaire – L1 Informatique
